Pre-requisites 
---------------
1. J2SE 1.7 (for Dev) JRE 1.7 for Just Run

2. MongoDb 2.4


Configuration:
---------------
At the bare minimum the following must be configured in src/main/webapp/WEB-INF/app.properties
	- eclipselink.nosql.property.mongo.port
	- eclipselink.nosql.property.mongo.host

Additional Optional Configuration 
1. General Logging config in log4j.properties src/main/resources/log4j.properties
